# Logos Directory

This directory should contain the logo assets for Plantão Fácil.

## Required Files

Place your logo files here with the following names:

### Main Logo
- `logo-main.svg` or `logo-main.png` - Logo principal para uso geral
- `logo-white.svg` or `logo-white.png` - Logo para fundos escuros
- `logo-small.svg` or `logo-small.png` - Logo compacto para mobile

### Email Logo
- `logo-email.png` - Logo para templates de email (PNG para melhor compatibilidade)

## Recomendações

- **Formato**: SVG é preferencial para logos na web (escala infinitamente)
- **Tamanho**: PNG não deve exceder 50KB (comprima com TinyPNG se necessário)
- **Dimensões sugeridas**:
  - Logo principal: 180x50px (ou proporção similar)
  - Logo pequeno: 40x40px
  - Logo email: 160x40px

## Fallback

Se os arquivos não forem fornecidos, a aplicação usará texto como fallback: "Plantão Fácil"
