# Public Assets - Plantão Fácil

Este diretório contém os assets públicos da aplicação.

## Favicon Files Needed

Coloque os seguintes arquivos de favicon na raiz de `public/`:

- `favicon.ico` - Favicon tradicional (16x16, 32x32, 48x48)
- `favicon.svg` - Favicon moderno em SVG
- `apple-touch-icon.png` - Ícone para dispositivos Apple (180x180px)

## Ferramentas Recomendadas

Para gerar favicons em múltiplos formatos:
- https://realfavicongenerator.net/
- https://favicon.io/

## Estrutura de Diretórios

```
public/
├── logos/           # Logos da aplicação (SVG/PNG)
├── favicon/         # Assets de favicon (se necessário múltiplos tamanhos)
├── favicon.ico      # Favicon principal
├── favicon.svg      # Favicon SVG
└── apple-touch-icon.png  # Ícone Apple
```
